## Описание проекта

В ходе лабораторной работы создано простое веб-API для управления списком любимых игр. Проект реализован на ASP.NET Core с использованием **Controller-based API** (контроллеры). Данные хранятся в памяти сервера (статический список). API поддерживает полный набор CRUD-операций (создание, чтение, обновление, удаление) и расширен дополнительными маршрутами для работы с избранными играми и валидацией.

---

## Функциональность

- **GET /api/games** — получить список всех игр
- **GET /api/games/{id}** — получить одну игру по ID
- **GET /api/games/favourites** — получить только избранные игры (самостоятельное задание)
- **POST /api/games** — добавить новую игру (с валидацией названия)
- **PUT /api/games/{id}** — обновить существующую игру
- **DELETE /api/games/{id}** — удалить игру

## Как запустить проект

### 1. Перейти в папку проекта
```bash
cd GamesApi
```

## 2. Восстановить зависимости
```bash
dotnet restore
```

## 3. Запустить сервер

```bash
dotnet run
```
